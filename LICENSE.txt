Computer Vision Code Gallery - maxEmbedded
==========================================

This repository is basically the Computer Vision Code Gallery of maxEmbedded. You can find all the codes for all the Computer Vision related posts here at one place. Codes are available for both OpenCV (C++ and Python) and MATLAB.
